# T4P (Tech for Players)

## Опис проєкту
T4P — це онлайн-магазин для продажу персональних комп’ютерів та конфігуратор для створення власних збірок.
Проєкт розроблено в рамках лабораторних робіт.

## Структура проєкту
- `index.html` - Головна сторінка
- `catalog.html` - Каталог товарів
- `contacts.html` - Сторінка контактів з формою
- `cart.html` - Кошик користувача
- `login.html` - Сторінка входу та реєстрації
- `admin.html` - Панель адміністратора
- `css/` - Папка зі стилями (style.css)
- `images/` - Папка із зображеннями
- `js/` - Папка для скриптів
